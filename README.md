\# QA Dashboard \& Employee Management API



\## 🚀 Project Overview



This is an Enterprise Spring Boot REST API project developed using Java and Spring Boot.



The project demonstrates enterprise backend development concepts including:



\- Employee Management

\- Student Management

\- Department Management

\- JWT Authentication

\- Spring Security

\- Swagger API Documentation

\- File Upload \& Download

\- Excel Import \& Export

\- Email Sending

\- MySQL Database

\- Docker

\- Docker Compose

\- Caching

\- Logging (Log4j2)



\---



\## 🛠 Technologies Used
